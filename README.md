# fzf-scripts

Scripts for fzf

## fzf-figlet

## Dependencies
- find
- sed
- fzf
- figlet
- toilet (optional)

## Use

1. Make executable
2. Put on `$PATH`
3. 
    ```sh
    fzf-figlet "<string>"
    ```

The string arg is displayed in the preview window in the selected font.
On confirmation of the selection, string arg is printed to the terminal
in that font.
